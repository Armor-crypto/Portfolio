Het tweede project ging over het gebruik van Javascript om games te maken. Met een editor genaamd "p5.js" is het mogelijk om een canvas te maken, waarop je Javascript kunt gebruiken om te tekenen. Hiernaast is een voorbeeld uit het lesboek weergegeven.

Ik maakte eerst verschillende programma's om te begrijpen hoe Javascript werkt. Hiernaast is een voorbeeld dat ik heb gebruikt om te begrijpen hoe je een object (een bal in dit geval) kunt controleren en hoe je ervoor kan zorgen dat het binnen de canvas blijft.

De laatste opdracht en mijn eindresultaat was het maken van een spel. Ik heb gekozen voor een javascript versie van ping pong, die ook in de achtergrond te zien is. Het combineert de kennis die ik uit andere opdrachten heb geleerd. Het spel werkt zoals normaal pong, en het heeft ook een kleurveranderingsanimatie voor de bal.
